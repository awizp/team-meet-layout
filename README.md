# Team Meet - Connect with anyone at anytime from anywhere through our service | Landing page

A clean and responsive landing page built using HTML, CSS and TailwindCSS for a Team Meet clone application. Designed with a modern layout, fast loading and mobile friendly.

### Features

* Responsive Design: The layout adapts to different devices, ensuring a great user experience on desktop, tablet, and mobile devices.

### Languages used

* HTML: The structure of the web page.

* CSS: Used for styling and media queries to make the design responsive.

* TailwindCSS: To fetch modern styles to clean look.

### Installation

1. Clone this repository to your local machine

``````git clone https://github.com/awizp/team-meet-layout.git``````

2. Open the ```index.html``` file in your browser to view the project.
